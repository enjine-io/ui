CheckboxGroup

Adds a CheckboxGroup to a given layout.

ckb = ui.addCheckboxGroup( parent, list, options, width, height )
ParametersTypeDescription
parent LayoutThe parent layout where to add the CheckboxGroup.
list ArrayThe items to be displayed.
options StringA comma separated options.
Theme Color: Default, Primary, Secondary
Sizes: Small, Medium
Icon Position: Left, Right
Container: Elevated, Outlined
Corners : Square
width NumberFraction of the parent width [0-1].
height NumberFraction of the parent height [0-1].

Returns CheckboxGroup Component

callback FunctionThe function to be called. This will pass value Boolean The checked state of the checkbox item. text String The checkbox label text. index Number The index of the corresponding checkbox. pos Object The position of the touch event.

Examples

EXAMPLE - BASIC

class Main extends App
{
	onStart()
	{
		// Creates a fullscreen layout with objects vertically centered.
		this.main = ui.addLayout( "main", "Linear", "VCenter", 1, 1 )

		var list = [
			"Checkbox item 1",
			"Checkbox item 2",
			"Checkbox item 3"
		]

		// Add a checkbox group to the main layout.
		this.ckb = ui.addCheckboxGroup( this.main, list, "", 0.9  )
	}
}

class Main extends App

EXAMPLE - CALLBACK

class Main extends App
{
	onStart()
	{
		// Creates a fullscreen layout with objects vertically centered.
		this.main = ui.addLayout( "main", "Linear", "VCenter", 1, 1 )

		var list = [
			"Checkbox item 1",
			"Checkbox item 2",
			"Checkbox item 3"
		]

		// Add a checkbox group to the main layout.
		this.ckb = ui.addCheckboxGroup( this.main, list, "Primary", 0.9 )
		this.ckb.setOnTouch( this.onChange )
	}

	onChange( text, checked, value )
	{
		ui.showPopup( text + " : " + checked )
	}
}
