ColorPicker

Shows a color picker. Customize a color or choose from the presets of color carefully selected from Material Design Color Pallette.

ui.showColorPicker( value, options, onSelect )
ParametersTypeDescription
value StringA hexadecimal default value for the color picker.
options StringA comma separated options.
Includes alpha: Alpha
onSelect FunctionThe callback function to be called when onselect event is fired. This will pass value String Color in hexadecimal format #rrggbb or #aarrggbb if Alpha option is passed.

Returns ColorPicker Component

Properties

Here are the setter and getter properties available for ColorPicker Component.

PropertiesTypeDescription
value StringReturns the value of the chosen color.

Methods

Here are the available methods of the ColorPicker Component.

getValue

Returns the hexadecimal string color value of the color picker.

cpk.getValue()

Returns String

Examples

EXAMPLE - BASIC

   class Main extends App
   {
       onStart()
       {
           this.main = ui.addLayout( "main", "Linear", "VCenter", 1, 1 )

           this.btn = ui.addButton( this.main, "Show Popup", "Primary" )
           this.btn.setOnTouch( this.ShowPopup )
       }

       ShowPopup()
       {
           ui.showColorPicker( null, "Alpha", this.onSelect )
       }

       onSelect( clr )
       {
           ui.showPopup( clr )
       }
   }
