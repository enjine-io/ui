Drawer

Adds a drawer into your app.

drw = ui.addDrawer( lay, options )
ParametersTypeDescription
layout LayoutThe drawer layout where to add controls.
options StringA comma separated options.nVariant: Permanent, Persistent, Temporary
Anchor: Left, Right, Top, Bottom
width NumberDistance from the anchor position [0-1]

Returns Drawer Component

Properties

Here are the available setter and getter properties of the Drawer Component.

PropertiesTypeDescription
anchor StringSets or returns the anchor position of the drawer. Values can be left, top, right, bottom.
disabled BooleanSets or returns the disabled state of the drawer.
swipeAreaWidth NumberSets or returns the width of the swipeable area of the Drawer.
width NumberSets or returns the width of the drawer as a fraction of the screen viewport relative to its anchor position [0-1].

Methods

Here are the available methods of the Drawer Component.

addLayout

Adds a layout into the drawer.

ParametersTypeDescription
lay LayoutLayout component.
disableSwipeToOpen

Disable swipe to open feature of the drawer. You can use this especially for ios devices which has swipe to go back feature.

drw.disableSwipeToOpen( value )
ParametersTypeDescription
value BooleanValues can be true or false.
disableBackdropTransition

Disable the backdrop transition. This can improve the FPS on low-end devices.

drw.disableBackdropTransition( value )
ParametersTypeDescription
value BooleanValues can be true or false.
disableDiscovery

If true, touching the screen near the edge of the drawer will not slide in the drawer a bit to promote accidental discovery of the swipe gesture.

drw.disableDiscovery( value )
ParametersTypeDescription
value BooleanValues can be true or false.
setOnOpen

Adds a callback function when the drawer is opened.

drw.setOnOpen(callback)
ParametersTypeDescription
callback FunctionThe callback function.
setOnClose

Adds a callback function when the drawer is closed.

drw.setOnClose(callback)
ParametersTypeDescription
callback FunctionThe callback function.
show

Open the drawer dynamically.

drw.show( anchor )
ParametersTypeDescription
anchor StringOptional. The anchor position. Can be left, top, right, bottom. Default is left.
hide

Close the drawer dynamically.

drw.hide()

Examples

EXAMPLE - DRAWER IMPLEMENTATION

   class Main extends App
   {
       onStart()
       {
           // Creates a fullscreen layout with objects vertically centered.
           this.main = ui.addLayout( "main", "Linear", "VCenter", 1, 1 )

           // Adds a text
           ui.addText(this.main, "<---- Swipe left ----->\n or", "Multiline,Center");

           // Adds a button to the main layout.
           this.btn = ui.addButton( this.main, "Open Drawer", "Contained,Primary" )
           this.btn.setOnTouch( this.openDrawer )

           // Creates the drawer layout.
           this.drawerLay = ui.addLayout( null, "Linear" )

           // Initialize the drawer by passing the drawer layout.
           this.drawer = ui.addDrawer( this.drawerLay, "left" )
           this.drawer.setOnClose( this.onClose )

           var lst1 = [
               [ "music_note", "Audios" ],
               [ "movie", "Videos" ],
               [ "insert_drive_file", "Documents" ]
           ]

           // Adds a list to the drawer layout.
           this.lstMenu1 = ui.addList( this.drawerLay, lst1, "Icon", 1 )
           this.lstMenu1.setOnTouch( this.closeDrawer )
           this.lstMenu1.label = "Main navigation"

           // Adds a divider into the drawer layout.
           ui.addDivider( this.drawerLay, 1 )

           var lst2 = [
               [ "mail", "All Mail" ],
               [ "inbox", "Inbox" ],
               [ "drafts", "Outbox" ],
               [ "send", "Sent" ]
           ]

           // Adds another list to the drawer layout.
           this.lstMenu2 = ui.addList( this.drawerLay, lst2, "Icon", 1 )
           this.lstMenu2.label = "Secondary navigation"
           this.lstMenu2.setOnTouch( this.onSelect )
       }

       openDrawer()
       {
           this.drawer.show()
       }

       onSelect(title)
       {
           ui.showPopup(title);
           this.drawer.hide()
       }

       onClose()
       {
           ui.showPopup('Drawer is close', "bottom")
       }
   }
EXAMPLE - DRAWER ANCHOR POSITIONS

   class Main extends App
   {
       onStart()
       {
           // Creates a fullscreen layout with objects vertically centered.
           this.main = ui.addLayout( "main", "Linear", "VCenter", 1, 1 )

           // Adds an appbar to the main layout
           this.apb = ui.addAppBar(this.main, "Drawers", "Menu")
           this.apb.setOnMenu( this.openDrawer )

           // Adds a select to the main layout.
           this.sel = ui.addSelect(this.main, ["Left", "Top", "Right", "Bottom"], "Radio", 0.3)
           this.sel.label = "Select anchor position"
           this.sel.setOnChange(this.onSelect)

           // Creates the drawer layout.
           this.drawerLay = ui.addLayout( null, "Linear" )

           // Initialize the drawer by passing the drawer layout.
           this.drawer = ui.addDrawer( this.drawerLay, "left" )

           var lst1 = [
               [ "music_note", "Audios" ],
               [ "movie", "Videos" ],
               [ "insert_drive_file", "Documents" ]
           ]

           // Adds a list to the drawer layout.
           this.lstMenu1 = ui.addList( this.drawerLay, lst1, "Icon", 1 )
           this.lstMenu1.setOnTouch( this.closeDrawer )
           this.lstMenu1.label = "Main navigation"

           // Adds a divider into the drawer layout.
           ui.addDivider( this.drawerLay, 1 )

           var lst2 = [
               [ "mail", "All Mail" ],
               [ "inbox", "Inbox" ],
               [ "drafts", "Outbox" ],
               [ "send", "Sent" ]
           ]

           // Adds another list to the drawer layout.
           this.lstMenu2 = ui.addList( this.drawerLay, lst2, "Icon", 1 )
           this.lstMenu2.label = "Secondary navigation"
           this.lstMenu2.setOnTouch( this.closeDrawer )
       }

       openDrawer()
       {
           this.drawer.show()
       }

       closeDrawer( title )
       {
           this.drawer.hide()
       }

       onSelect( value )
       {
           // Set the drawer anchor first
           this.drawer.anchor = value
           // Set the drawer width depending on anchor position
           if(value == "Top" || value == "Bottom") {
               this.drawer.width = 0.5
           } else {
               this.drawer.width = 0.7
           }
           // Open the drawer
           this.drawer.show()
       }
   }
