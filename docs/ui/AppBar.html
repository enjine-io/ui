AppBar

Adds an AppBar on your app.

apb = ui.addAppBar( parent, title, options, width, height )
ParametersTypeDescription
parent LayoutThe parent layout where to add the AppBar
title StringThe title text of the appbar
options StringA comma separated options.
Menu icon: Menu
Theme Color: Primary, Secondary, Transparent, Inherit, Default
Position: Absolute, Static, Fixed, Relative
width NumberFraction of the screen width [0-1]
height NumberFraction of the screen height [0-1]

Returns AppBar Component

Methods

Here are the methods available for AppBar Component

setOnTouch

Adds a callback handler when the component is touch.

cname.setOnTouch( callback )
ParametersTypeDescription
callback FunctionThe callback function to be called. This will pass pos Object The position of the touch event.
setOnMenu

Sets a function to be called when the user clicks the menu

apb.setOnMenu( callback )
ParametersTypeDescription
callback FunctionThe function to be called. This will pass pos Object The position of the touch event.
addLayout

Adds a layout for additional controls at the right of the appbar

apb.addLayout( lay )
ParametersTypeDescription
lay LayoutThe layout where to add controls
setCornerRadius

Sets the corner radius of the appbar.

acc.setCornerRadius(tl, tr, bl, br, mode)
ParametersTypeDescription
tl NumberTop-left corner radius.
tr NumberTop-right corner radius.
bl NumberBottom-left corner radius.
br NumberBottom-right corner radius.
mode StringUnit. Values are px rem or %.

setOnContextMenu

Adds a callback function on right click.

setOnContextMenu( callback )
ParametersTypeDescription
callback FunctionThe callback function to be called on context menu event or right click event. This will pass event Object The pointer event object.

apb.hide()

setScale

Sets the x and y scaling of the component.

apb.setScale( x, y )
ParametersTypeDescription
x NumberThe x-scale of the component.Values less than 0 is smaller than the normal. While values greater than 1 is greater than the normal.
y NumberThe y-scale of the component. Values less than 1 is smaller than the normal. While vaues greater than 1 is greater than the normal.
getPosition

Returns the position of the component. The return object is of the form {left, top, right, bottom}

apb.getPosition( options )
ParametersTypeDescription
options StringThe mode of the measurements. Values can be px or %

Returns Object

setMargins

Sets the margin of the component.

apb.setMargins( left, top, right, bottom, mode )
ParametersTypeDescription
left NumberLeft margin. You can also pass string e.g. 12px
top NumberTop margin. You can also pass string e.g. 12px
right NumberRight margin. You can also pass string e.g. 12px
bottom NumberBottom margin. You can also pass string e.g. 12px
mode StringUnit of measurement.
px for pixels
% relative to its parent dimension.
v relative to viewport dimension.

Sets the padding component container.

apb.setPadding(left, top, right, bottom, mode)
ParametersTypeDescription
left NumberFraction of the component width.
top NumberFraction of the component height. [0-1]
right NumberFraction of the component width. [0-1]
bottom NumberFraction of the component height. [0-1]
mode StringThe size thickness mode. Can be px
setPosition

Sets the position of the component relative to its parent dimensions.

apb.setPosition( left, top, mode )
ParametersTypeDescription
left NumberFraction of the parent width. [0-1]
top NumberFraction of the screen height. [0-1]
mode StringUnit of measurement. Can be px or % or any css unit of measurement.
setBorder

Sets the border line for the component container.

apb.setBorder( left, top, right, bottom, clr )
ParametersTypeDescription
width NumberBorder-left thickness in pixels.
clr StringBorder color in hexadecimal form #rrggbb
style StringBorder-styles. Values can be dotted, dashed, solid, double, groove, ridge, inset and outset. Default is solid
bringForward

Bring this component forward by a given z-index.

apb.bringForward( zIndex )
ParametersTypeDescription
zIndex NumberThe z-index. A negative value behaves like sendBackward method.
sendBackward

Bring this component backward by a given z-index.

apb.bringForward( zIndex )
ParametersTypeDescription
zIndex NumberThe z-index. A positve value behaves like bringForward method.

Examples

EXAMPLE - BASIC APPBAR

class Main extends App
{
	onStart()
	{
		// Creates a fullscreen layout with objects vertically centered.
		this.main = ui.addLayout( "main", "Linear", "VCenter", 1, 1 )

		// Adds an appbar to the layout
		this.apb = ui.addAppBar( this.main, "My App", "", 1 )
	}
}
EXAMPLE - APPBAR WITH MENUS

   class Main extends App
   {
       onStart()
       {
           // Creates a fullscreen layout with objects vertically centered.
           this.main = ui.addLayout( "main", "Linear", "VCenter", 1, 1 )

           // Adds an appbar to the layout
           this.apb = ui.addAppBar( this.main, "My app", "Menu,Fixed", 1 )
           this.apb.setOnMenu( this.showDrawer )

           // Adds a drawer layout
           this.drawLay = ui.addLayout(null, "Linear", "Top")

           // Adds a drawer to the app and pass the drawer layout
           this.drawer = ui.addDrawer(this.drawLay, "left")

           // Adds a list to the drawer layout
           let lst = [
               ["folder", "Folders"],
               ["music_note", "Audios"],
               ["photo", "Photos"]
           ]
           this.lstMenu = ui.addList(this.drawLay, lst, "Icon", 1 )
           this.lstMenu.label = "Main navigation"
       }

       showDrawer() {
           this.drawer.show()
       }
   }
EXAMPLE - APPBAR WITH CONTROLS

   class Main extends App
   {
       onStart()
       {
           // Creates a fullscreen layout with objects vertically centered.
           this.main = ui.addLayout( "main", "Linear", "Top,Center", 1, 1 )

           this.apb.setOnMenu( this.showDrawer )
       
           // Add an icon buttons to the appbar
           this.btn1 = ui.addButton( this.apb.layout, "person", "icon" )
           this.btn2 = ui.addButton( this.apb.layout, "delete", "icon" )

           // Adds a drawer to the app and pass the drawer layout
           this.drawer = ui.addDrawer(this.drawLay, "left")

           // Adds a list to the drawer layout
           let lst = [
               ["folder", "Folders"],
               ["music_note", "Audios"],
               ["photo", "Photos"]
           ]
           this.lstMenu = ui.addList(this.drawLay, lst, "Icon", 1 )
           this.lstMenu.label = "Main navigation"
       }

       showDrawer() {
           this.drawer.show()
       }
   }
EXAMPLE - APPBAR WITH SEARCH

   class Main extends App
   {
       onStart()
       {
           // Creates a fullscreen layout with objects vertically centered.
           this.main = ui.addLayout( "main", "Linear", "Top", 1, 1 )

           this.apb = ui.addAppBar( this.main, "My app", "Default" )
       
           // Adds a textfield to the appbar
           this.tfd = ui.addTextField(this.apb.layout, "", "Search,Outlined,Small")
           this.tfd.placeholder = "Search"
       }
   }
