Switch

Adds a Switch Component into your layout. A switch is a control whose values can be toggled between on and off.

swt = ui.addSwitch( parent, text, options, width, height )
ParametersTypeDescription
parent LayoutThe parent layout where to add the Switch Component.
text StringThe label text.
options StringA comma separated options.
Theme Color: Primary, Secondary, Default
Icon Position: Left, Top, Right, Bottom
Sizes: Small, Medium
width NumberFraction of the screen width [0-1]
height NumberFraction of the screen height [0-1]

Returns Switch Component

Methods

Here are the available methods of the Switch Component.

setOnTouch

Adds an event handler when the switch is touch.

swt.setOnTouch( callback )
ParametersTypeDescription
callback FunctionThe callback function to be called. This will pass check Boolean The checked state of the switch control.

setOnContextMenu

Adds a callback function on right click.

setOnContextMenu( callback )
ParametersTypeDescription
callback FunctionThe callback function to be called on context menu event or right click event. This will pass event Object The pointer event object.

Examples

EXAMPLE - BASIC

   class Main extends App
   {
       onStart()
       {
           // Creates a layout with objects verticaly centered.
           this.main = ui.addLayout( "main", "Linear", "VCenter", 1, 1 )

           // Adds a switch to the main layout.
           this.swt = ui.addSwitch( this.main, "Check me", "Secondary", 0.5 )
           this.swt.setOnTouch( this.onTouch )
       }

       onTouch( value )
       {
           ui.showPopup( "Value is : "  + value )
       }
   }
